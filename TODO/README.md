1. Instal·lar Django: pip install django


 2. Crear projecte: django-admin startproject nom_projecte


 3. Crear app: python manage.py startapp nom_app


 4. Registrar app a settings.py → INSTALLED_APPS


 5. Crear model a models.py


 6. Fer migracions: python manage.py makemigrations


 7. Aplicar migracions: python manage.py migrate


 8. Crear superusuari: python manage.py createsuperuser


 9. Registrar model a admin.py


 10. Crear view a views.py


 11. Crear template a templates/app/


 12. Configurar URLs a urls.py


 13. Executar servidor: python manage.py runserver


 14. Provar l'aplicació al navegador
